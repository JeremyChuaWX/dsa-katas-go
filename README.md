## How to use

- To test the katas, run the command

  ```bash
  go test ./katas_test
  ```

- To generate stubs from template files, run the command

  ```bash
  go run ./templater.go --name <name-of-kata>
  ```

  Add the `--force` flag to forcefully reset existing katas
